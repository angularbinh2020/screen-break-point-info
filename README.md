# About

Use to display screen break down label to easy responsive style

# Usage

breakPoints is an object : key will be use to display break point label, value is min screen width


```
 <ScreenBreakPointInfo
          breakPoints={{
            sm: 640,
            md: 768,
            lg: 1024,
            xl: 1280,
            "2xl": 1536,
          }}
        />
```
